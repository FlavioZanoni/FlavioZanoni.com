## **The company:**

SmartBr is the technological branch of an physiotherapist equipment maker and marketplace in Brazil, that aims to develop its ecommerce, its own ERP for the physiotherapist business and the new ERP/CRM being developed aimed for beauty salons and clinics, developing hardware and software to better integrate the equipment and de software

## **My role:**

As a Front End Developer, I played a key role in designing and developing features for the ERP. Utilizing Next.js, React.js, and other technologies, I contributed to enhancing the application's user interface and experience. My involvement extended to exploring and integrating new technologies to improve system efficiency and functionality. Additionally, I actively participated in the system design process, offering insights and suggestions to optimize the application's performance.

- \- Built server-side rendered applications with Next.js and React.js to achieve faster load times and better SEO.
- \- Utilized tech stacks in Next.js, React.js, Axios, Typescript and React-Query to build an ERP system for beauty salons and health clinics with a micro frontend design
- \- Implemented data fetching and caching using Axios and React-Query for efficient handling of operational needs.
- \- Documented system architecture, designs, and implementation details for reference and future development.
- \- Conducted code reviews and implemented necessary improvements as well as help with the incomplete type system as the project was being migrated to typescript.
